# Low Battery Delivery

## Overview
A 2D Godot game based on the theme **“Running Out of Power.”**  
You control a drone that must deliver a package before its battery runs out.

## Controls
- Move: WASD / Arrow Keys  
- Boost: Shift (drains battery faster)  
- Restart: R  

## Gameplay
- Battery drains while moving  
- Collect batteries to recharge  
- Avoid hazards that reduce battery  
- Pick up the package and deliver it to the goal
